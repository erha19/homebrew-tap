# Erha19 Tap

## How do I install these formulae?

`brew install erha19/tap/<formula>`

Or `brew tap erha19/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "erha19/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
